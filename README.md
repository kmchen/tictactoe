# Based on React Redux minimal starter kit (boilerplate)

# Getting started

1. Open a command prompt (terminal) and type in the following:

    ```sh
    git clone https://github.com/kmchen/tictactoe
    npm install
    npm start
    ```

2. Then open your [http://localhost:8080/]

3. To run test:

    ```sh
    npm test
    ```
